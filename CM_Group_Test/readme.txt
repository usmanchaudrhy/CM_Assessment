** Development Environment **
* All Questions are answered using Node js version 14.15.4 on a Mac Operating System

run - npm install - to install all the additional packages required

** Testing Framework **

* Testing framework used is Mocha and Chai, and all test files are in the test folder, it
    contains tests for Q1,Q2,Q3,Q4 and Q7
* Test file contains tests for all the sample inputs and expected outputs as listed in the document
* Tests can be run by running the command - npm run test

** Answers to the Questions **

* Question 1 to 4 are answered in the file answers.js
* Question 5 and 6 are SQL questions, in the file SQL.txt, the queries have been run in MySQL Workbec
* Question 7 is in the file q7.ans.js -> its has been implemented as a class, the constructor to the class
    initializes it with the key on which to group the objects

** You can see the output for the sample inputs provided **

* npm run test -> will run all the tests on questions 1,2,3,4 and 7
* node index.js -> will run Question 1,2,3,4 and 7 and output all the results on the console
* node q8.ans.js -> will run Question 8 on the input file test.html, parse it and show its result on the console
